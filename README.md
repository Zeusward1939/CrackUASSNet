# CrackUASSNet
Uncertainty-aware pseudo-label learning for enhanced semi-supervised crack detection with limited and noisy labels

High-quality pixel-level annotations for concrete crack detection are labor-intensive and costly. While semi-supervised learning mitigates data dependency, existing methods often suffer from unreliable pseudo-labels and confirmation bias due to complex backgrounds and diverse crack morphologies. Moreover, their robustness is frequently compromised by out-of-distribution data and noisy labels. To address these issues, we propose CrackUASSNet, an uncertainty-aware pseudo-label learning framework designed for enhanced semi-supervised crack detection. The proposed network architecture features a shared contracting path integrated with multi-branch perturbation expansive paths to efficiently introduce latent space stochasticity. To mitigate overfitting to noisy predictions, a dynamic stochastic label fusion mechanism is utilized to generate highly robust pseudo-labels. Furthermore, the model employs Kullback-Leibler divergence to quantify pixel-wise prediction uncertainty, using the resulting uncertainty maps to dynamically weight the pseudo-label supervision loss and actively suppress noise interference. Extensive experiments on three datasets (DeepCrack, Crack776, and Crack1242) demonstrate that with only 20% labeled data, CrackUASSNet achieves segmentation accuracy comparable to state-of-the-art fully-supervised learning methods. Also, the uncertainty-guided regularization mechanism effectively mitigates confirmation bias, enabling the model to exhibit superior robustness and generalization against both out-of-distribution data and significant label noise. Consequently, CrackUASSNet provides a highly reliable and cost-effective solution for automated structural inspections.

# Useage
If you want to train the model, run the following command in the terminal:
```
cd CrackUASSNet
python main/UASS_train/UASS_train.py
```
If you want to validate the model and generate images, run the following command in the terminal:
```
cd CrackUASSNet
python main/test.py
```
